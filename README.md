# Aplikacja do układania tangramów / Tangram puzzle app

(PL) Aplikacja do układania tangramów napisana w języku PHP oraz w HTMLu (+CSS) i JavaScripcie. Jako baza danych dla wspólrzędnych figur wykorzystałem bazy danych SQL. Do projektu użyłem również frameworku Konva.js m.in. do rysowania kształtów oraz ich dopasowywania. Zaimplementowałem 3 kategorie tangramów: ludzie, zwierzęta oraz rzeczy, co daje łącznie 16 wzorów tangramów. Dla poprawy wyglądu dodałem również funkcję zmiany motywu (jasny, ciemny). Kształty można obracać (klikając prawym przyciskiem myszy) oraz przesuwać (trzyamjąc je lewym przyciskiem myszy). Aplikacja została również przystosowana do działania na systemie operacyjnym iOS (za sprawą zmienionej funkcjonalności prawego przycisku myszy na tym systemie).

(EN) A tangram-assembling application written in PHP, HTML (+CSS), and JavaScript. I used SQL databases as the database for figure coordinates.. I also used the Konva.js framework for drawing and aligning shapes, among other things. I implemented three tangram categories: people, animals, and things, for a total of 16 tangram patterns. To improve the appearance, I also added a theme changer (light or dark). Shapes can be rotated (by right-clicking) and moved (by holding down the left mouse button). The application has also been adapted to run on iOS (thanks to the modified right-click functionality on iOS).
